# ERP/CRM/POS System

## AMS (Açık ve Modern Sistem)
- Eğitimciyiz, birden fazla sektöre hizmet veren ERP, CRM ve POS sistemleri geliştirdik.

## Özellikler Listesi
- Tam entegre ERP, CRM ve POS fonksiyonları
- Kullanıcı dostu arayüz
- Esnek raporlama
- Çok kiracılı mimari

## Teknoloji Yığını
- .NET 5
- MS SQL Server
- Entity Framework Core
- JavaScript ve jQuery

## Kurulum Talimatları
1. **Gereksinimler**: .NET 5 SDK ve MS SQL Server.
2. **Projeyi Klonlayın**: `git clone https://github.com/alfateknoloji45-glitch/ErpCrmSystem.git`
3. **Bağımlılıkları Yükleyin**: `dotnet restore`
4. **Veritabanını Oluşturun**: Aşağıdaki SQL dosyasını çalıştırın.
